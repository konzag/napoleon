# ⭐ Η Προγραμματιστική Οδύσσεια του Ναπολέων ⭐

> *«Πριν πολύ καιρό σε έναν κήπο μακριά, μακριά...»*

Καλωσήρθες στην **Προγραμματιστική Οδύσσεια του Ναπολέων** — την κωδικιστική περιπέτεια ενός νεαρού Jedi Padawan που αγαπά τα ζώα 🐾, τα φυτά 🌿, τους δεινόσαυρους 🦕, τη μουσική 🎵, τα LEGO 🧱, και το Star Wars ⚔️!

Ο Ναπολέων είναι **6 χρονών** και αυτό είναι το πρώτο του project προγραμματισμού. Κάθε αρχείο εδώ είναι γραμμένο με πολλά σχόλια ώστε ο Ναπολέων να μπορεί να τα διαβάσει κάποια μέρα και να καταλαβαίνει ακριβώς πώς λειτουργούν.

## 🚀 Projects

### 1. 🌐 Εγκυκλοπαίδεια Ζώων, Φυτών & Δεινόσαυρων (`index.html`)

Ένα **διαδραστικό παιχνίδι στο διαδίκτυο** — άνοιξέ το σε οποιονδήποτε browser, χωρίς εγκατάσταση!

**Τι κάνει:**
- 🐾 Κλίκαρε σε οποιοδήποτε ζώο για να δεις ένα διασκεδαστικό γεγονός και να ακούσεις ήχο lightsaber
- 🌿 Πήγαινε στην καρτέλα Φυτά — κάθε κάρτα παίζει έναν ήχο φύσης chime
- 🦕 Εξερεύνησε την καρτέλα Δεινόσαυροι με 6 προϊστορικά πλάσματα και ήχο βρυχηθμού
- 🧠 Δοκίμασε το Quiz Mode — 8 ερωτήσεις, Jedi ranking στο τέλος!
- 🧱 Χτίσε με το LEGO Builder — 10×8 πλέγμα με 8 χρώματα LEGO
- 🎵 Βελτιωμένοι ήχοι: lightsaber hum, nature chime, R2-D2 beeps, droid malfunction, whoosh
- ⚔️ Σχεδιασμός με θέμα Star Wars με χρώματα φωτόσπαθου και φόντο διαστήματος
- 📱 Λειτουργεί και σε κινητά τηλέφωνα και τάμπλετ

### 🧠 Quiz Mode *(ΝΕΟΝ!)*

- **8 ερωτήσεις** βασισμένες στα γεγονότα των ζώων και φυτών
- **4 επιλογές** ανά ερώτηση
- **Jedi Ranking** στο τέλος:
  * 🌟 8/8 → **Grand Master Jedi!**
  * ⚔️ 6-7 → **Jedi Knight!**
  * 💚 4-5 → **Jedi Padawan!**
  * 💚 0-3 → **Young Padawan, συνέχισε να μαθαίνεις!**
- Ήχοι: R2-D2 beeps για σωστό, droid malfunction για λάθος

---

### 🧱 LEGO Builder *(ΝΕΟΝ!)*

- **Πλέγμα 10×8** = 80 κελιά για χρωματισμό
- **8 χρώματα LEGO**: Κόκκινο, Μπλε, Κίτρινο, Πράσινο, Λευκό, Μαύρο, Πορτοκαλί + Γόμα
- Κάθε κελί μοιάζει με τουβλάκι LEGO (στρογγυλό stud από πάνω!)
- **Κλικ & σύρσιμο** για γρήγορο ζωγράφισμα
- Κουμπί **Καθαρισμός** για νέο σχέδιο
- Λεζάντα: *«Τι χτίζεις σήμερα, Padawan Napoleon?»*

| Εναλλαγή καρτέλας | 💨 Whoosh effect |

### 2. 🐍 Jedi Εξερευνητής Φύσης (`napoleon.py`)

Ένα **πρόγραμμα τερματικού** γραμμένο σε Python — η πρώτη γλώσσα προγραμματισμού του Ναπολέων!

**Τι κάνει:**
- 🛸 Εμφανίζει ένα ASCII διαστημόπλοιο και καλωσορίζει τον Ναπολέων ως Jedi Εξερευνητή Φύσης
- 📋 Ένα απλό αριθμημένο μενού με επιλογές
- 🐾 Μάθε για 8 καταπληκτικά ζώα
- 🌿 Μάθε για 5 απίστευτα φυτά
- 🎵 Παίξε μουσικές νότες — Ντο Ρε Μι Φα Σολ Λα Σι Ντο — ή το θέμα Star Wars!

**Πώς να το τρέξεις:**
```bash
python napoleon.py
```
